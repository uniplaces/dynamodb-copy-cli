## Development
To build and run this cmd, you'll need go (1.11.x) with mod support enabled `export GO111MODULE=on`

### Running
To run:
```
go run cmd/main.go
```
Alternatively, you can easily build the binary by:
```
go build -o dynamodbcopy cmd/main.go
```
